# Stable Ethereum Token

Proposed implementation for an stable token.

### Prerequisites

* npm
* Ganache
* Truffle v4.1.14

### Installing

Install NPM, Ganache and truffle.
Run Ganache, then:

```
truffle migrate
```

## Running the tests

```
truffle test
```
